
# Quantum Ai Labs

Front end ui development of a full stack web application

## Technology Stack

- React JS
- TypeScript
- Tailwind CSS
- Docker

### Libraries Used

 - React Fast Marquee
 - Swiper
 - React Spinners
 - Framer motion


 
## Run

Run Docker desktop

Run below two commands in a terminal

- docker build -t quantum:dev .


- docker run -p 5173:5173 quantum:dev
